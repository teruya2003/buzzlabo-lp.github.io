# buzzlabo-lp

BUZZ LABOのランディングページ

## 料金プラン

### 基本情報
- 2つの有料プラン（Growth、Agency）を提供
- どちらのプランも**14日間無料トライアル**あり
- クレジットカード登録後、14日以降に自動課金となります

### Growthプラン
**月額: ¥2,980**

**対象:** 個人事業・小チーム向け

**機能:**
- 連携アカウント: 5アカウント
- 取得履歴: 無制限
- 勝ちパターン保存: 30投稿
- 競合トラッキング: 5件
- メンバー数: 3名
- 共有レポート: PDF・CSV・スライド

### Agencyプラン
**月額: ¥9,800**

**対象:** 代理店・多店舗運用向け

**機能:**
- 連携アカウント: 20アカウント (拡張可)
- 取得履歴: 無制限
- 勝ちパターン保存: 無制限
- 競合トラッキング: 無制限
- メンバー数: 無制限
- 共有レポート: PDF・CSV・スライド
- カスタムダッシュボード: 複数作成可能

### 無料トライアルについて
- どちらのプランも14日間無料でお試しいただけます
- クレジットカード登録が必要です
- 14日間は料金は発生しません
- 14日以降に自動課金となります
